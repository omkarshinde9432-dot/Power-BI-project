## üìù **Short Description / Purpose**
- This project presents a comprehensive **On-Time In-Full (OTIF) performance analysis dashboard** built in Power BI.
- The goal of this dashboard is to help **supply chain, logistics, and operations teams** monitor key delivery metrics, identify performance gaps, and gain deeper insights into **city, customer, and product-level trends**.
- It supports **data-driven decision-making** by offering interactive drill-downs and clear KPIs aligned with business expectations.
